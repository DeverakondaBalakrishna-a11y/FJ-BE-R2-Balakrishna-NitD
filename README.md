# 💰 Personal Finance Tracker

A Django-based web application to help users track income, expenses, budgets, financial goals, and visualize data through interactive dashboards. Includes social login with Google, receipt uploads, notifications, and Render deployment.

---

## 🚀 Features

- ✅ User authentication (email/password & Google OAuth)
- ✅ Dashboard showing income, expenses, savings (with charts)
- ✅ Add, edit, and delete transactions
- ✅ Monthly and yearly reports
- ✅ Budget tracking with category-wise limits
- ✅ Budget overrun alerts via email (SendGrid)
- ✅ Upload receipts with transactions
- ✅ Set financial goals and track progress
- ✅ Render deployment-ready

---

## 📦 Tech Stack

- **Backend:** Django 4+
- **Database:** PostgreSQL (prod) / SQLite (dev)
- **Frontend:** HTML + Tailwind CSS 
- **Auth:** `django-allauth` with Google OAuth
- **Deployment:** Render
- **Email:** SendGrid SMTP
